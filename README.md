## Getting Started
1. Clone the repo
2. add OPENAI_API_KEY="yourkey" to .env

 ```
pip install sounddevice soundfile numpy playsound pillow opencv-python openai pyautogui
Issues: speech_recognition
python screenshare.py
python main.py
```
